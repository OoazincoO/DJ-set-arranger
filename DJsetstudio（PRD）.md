? 产品需求文档（PRD）
项目名称：智能歌单生成与 DJ Set 展示平台  
版本：V1.3（整合版）  
编写人：坚铭  
日期：2025-12

1. 产品概述
1.1 产品定位
本产品是一个跨平台的智能音乐管理平台，提供：

智能歌单生成

相似歌曲推荐

DJ Set 创建与展示

在线 Web 网站访问

Android 原生 App 支持

目标用户包括 DJ、音乐创作者、电子音乐爱好者。

1.2 产品愿景
让每一个热爱音乐的人，都能轻松构建属于自己的音乐世界。

1.3 产品目标
提供专业级的智能推荐能力

提供便捷的 Set 管理工具

提供跨平台的音乐展示体验

提供可扩展的音乐创作生态基础

2. 用户分析
2.1 用户角色
? DJ / 音乐创作者
需要快速找到相似曲目

需要管理自己的 Set

需要展示作品集

? 音乐爱好者
想快速生成歌单

想探索相似风格的音乐

? 移动端用户
想随时查看自己的 Set

想在演出现场快速查阅曲目

2.2 用户痛点
痛点	描述
找歌效率低	需要在多个平台搜索、筛选
歌单不够专业	缺少 BPM、风格、相似度等专业维度
Set 管理混乱	没有统一的管理工具
展示渠道少	作品难以展示给他人
2.3 用户价值
一键生成专业歌单

统一管理曲库与 Set

展示个人作品

提升创作效率

3. 产品功能需求（详细）
? 3.1 歌曲管理模块
3.1.1 歌曲搜索
支持关键词搜索（歌名、艺术家）

支持风格、BPM 筛选

搜索结果展示：封面、名称、艺术家、BPM、风格

3.1.2 添加歌曲到曲库
用户可将搜索结果加入个人曲库

系统需检查重复添加

曲库支持分页、排序

3.1.3 曲目详情页
展示歌曲信息

展示相似歌曲推荐入口

? 3.2 智能推荐模块
3.2.1 相似歌曲推荐
输入一首歌 → 返回 5–20 首相似歌曲

相似度基于 embedding 或标签体系

推荐结果可加入曲库或加入 Set

3.2.2 智能歌单生成
输入主题（如 “Future House 热门歌单”）

输入参考歌曲

系统自动生成歌单

用户可手动调整顺序

? 3.3 DJ Set 管理模块
3.3.1 创建 Set
填写名称、描述、封面

从曲库中选择歌曲

支持拖拽排序（Web）

3.3.2 编辑 Set
增删歌曲

修改顺序

修改封面与描述

3.3.3 展示 Set
Web：卡片式展示

Android：移动端适配展示

支持分享链接（未来版本）

? 3.4 用户系统
3.4.1 注册 / 登录
邮箱 + 密码

密码加密存储

登录返回 JWT Token

3.4.2 用户资料
昵称

头像（未来版本）

个人 Set 列表

? 3.5 跨平台支持（整合版）
3.5.1 Web 端（在线网站）
部署在服务器上

用户通过域名访问

使用 HTTPS 调用后端 API

示例：

Code
https://djset.app
3.5.2 Android 原生 App
真机安装 APK

使用 Retrofit 调用线上 API

与 Web 端共享账号体系

4. 非功能需求
4.1 性能
搜索响应时间 < 1 秒

推荐响应时间 < 2 秒

支持 1000+ 曲库数据

4.2 安全
密码加密

Token 鉴权

防 SQL 注入

4.3 可用性
UI 简洁

操作流畅

支持深色模式（未来）

4.4 可扩展性
推荐算法可替换

数据库可升级

支持更多音乐平台数据源
